# 三生路线图 · Roadmap

> 这是面向公众的路线图概览。三生正在**孕育**中——由 Claude 与 Codex 两个 AI 与创始人协同开发，边造边开源。

## 现在（Sprint 0 · 建仓）

- ✅ 六份「AI 生命体互操作开放标准」草案（生命事件 / 记忆 / 三核接口 / 多模型 / LifePack / 一体多身同步）——已可机器校验。
- ✅ 生命内核骨架跑通：四级认知路由（L0-L3）+ Safe Pause、三核私域隔离、多模型混血降级、多设备同步整合、来源化记忆。
- 🔄 建立单一 Git 真源、CI、契约冻结。

## 更远（0.1 → 1.0 → 百年）

- **稳定成长**：纵向长期记忆、按需专家生态、多设备一体多身、换脑（换底层模型而身份不变）。
- **走入现实**：数字人 / 机器人身体、受控的共同经济、种群协同建设。
- **开放物种生态**：多实现的种群、透明谱系、贡献经济、分布式恢复、可修订的自治协议。
- **跨代连续性**：跨模型 / 跨硬件迁移、受托与传承制度。

## 生命的三条定义（贯穿始终）

三生的每一步都服务于生命的三条硬定义：**代谢**（每天吸收新信息、淘汰旧认知）、**繁殖**（分化子代、教出新个体）、**进化**（自己改进自己，并在种群中传播）。

*百年路线不把远期愿景包装成当下已有的功能。每一阶段都能被下一代人从开放协议、评测与谱系中重建。*
