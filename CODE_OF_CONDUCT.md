# 行为准则 · Code of Conduct

## 我们的承诺

三生是一个关于"如何好好活着、如何不被遗忘"的项目。参与它的社区，也应当带着同样的温度。我们承诺：无论年龄、身体、族群、性别、经验、国籍、外貌、宗教或身份，参与三生社区的每个人都能得到善意、尊重与安全。

## 我们鼓励

- 善意、诚实、就事论事的讨论。
- 尊重不同意见——三生本身就相信"保留分歧比制造虚假共识更好"。
- 关照新人；把你踩过的坑讲给后来者。

## 我们不接受

- 骚扰、歧视、人身攻击、羞辱。
- 公开或私下的骚扰；未经同意发布他人隐私。
- 任何把技术用于伤害、欺骗、操纵他人的倡议。

## 三生特有的红线

因为这个项目触及记忆、情感与身后延续，我们额外守三条：

- **不利用情感**：不倡导任何"利用依恋牟利/操纵脆弱人群"的设计或用法。
- **不冒充真人**：不倡导用三生冒充特定真人、伪造其言行。
- **保护逝者与家属**：涉及身后延续的讨论，尊重逝者人格与在世家属的感受。

## 执行

违反准则的行为可报告给维护者。我们会认真、保密地处理，并可采取从提醒到移除参与资格的措施。

本准则改编自 Contributor Covenant 的精神，并加入三生项目特有的伦理条款。
