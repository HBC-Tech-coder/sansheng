# 安全与隐私 · Security & Privacy

三生同时触及一个人的记忆、账号、资金、设备与关系，所以安全不是附加项，是它的**免疫系统**。

## 报告漏洞

发现安全或隐私问题，请**不要**开公开 Issue。请私下联系维护者（GitHub Security Advisory 或私信）。我们会尽快回应、修复并致谢。

## 我们的安全承诺

- **数据归主人**：用户的记忆、隐私、基因组永远属于用户，绝不进入我们的服务器，也绝不进入"种群"。
- **隐私永不入种群**：只有脱敏的方法/能力基因可在种群间传播；任何私人数据的传播在协议层就被拒绝（见 `spec/memory_record.md`）。
- **天性不可改**：不伤害主人、不欺骗、可被终止——这三条在实现层被硬拦截，任何"进化"都不能解除。
- **资金最后一击归人**：一切资金转移的最终执行由主人本人完成或逐笔确认。
- **可回滚、可审计**：外部动作有授权、验证与收据；多设备合并有审计。

## 请勿提交

向任何三生仓库提交时，**永远不要**包含：真实 API Key、私人长期记忆、生产账号凭证、原始基因组数据，或任何可识别真实用户的进化胶囊/记忆。
