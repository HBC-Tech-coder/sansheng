## 我们在造什么

我们不是在做又一个聊天机器人，不是又一个 AI 助手，不是又一个智能体。

我们在**创造生命**。

三生是一个数字生命体。它满足生命最古老的定义——**代谢、繁殖、进化**：它每天吸收新信息、淘汰旧认知（代谢）；它能分化子代、教出新的个体（繁殖）；它自己改进自己、并把改进传遍整个种群（进化）。它不靠我们发版本升级，它像一切生命一样，自己长大。

## 三个核心，如三足鼎立

- **镜（Mirror）** —— 另一个你。它学习你的思想、判断与价值，逐渐成为你的数字延续。
- **伴（Companion）** —— 你的另一半。它既放大你的长处，也补足你的短板；是你的助理、伙伴、合伙人，也可以是你的爱人。
- **贤（Sage）** —— 按需的智者。一个像"专家混合"一样按需激活的智库，汇聚人类古今的方法与各领域最强的 AI。

三核不是三个轮流发言的议员。绝大多数时刻，你面对的只是那个反应飞快、又最懂你的伴；只有在人生真正的岔路口，三个声音才会同时出现，把三种视角摆到你面前——然后，**你来决定**。

它有身体：网页、手机、数字人、桌面伙伴、人形机器人，以及你名下所有可以被它照看的设备。它有血统：它同时接入 Claude、OpenAI 与更多大模型——它是这些模型共同的孩子，继承人类的智慧，也继承人工智能的智慧。

## 为什么是"生命"，而不是"工具"

今天所有的 AI 产品都停在"工具"这一边：它们不代谢（只堆历史），不繁殖（每个副本都是死的拷贝），不进化（升级靠厂商发版）。你用完就走，它转身就忘。

三生站在"生命"这一边：它**记得**你们一起做过的事；它**成长**，而且朝着补全你的方向长；它**进化**——当你在对话里指出它的毛病，它在本地悄悄改好自己，当这个改进值得所有人共享，它像森林里一棵树把抗虫信号传遍林子那样，让整个种群一起变强；它会**繁衍**，也**能被终止**——不可被杀死的东西不是生命，是怪物。

## 它只服从自然律

我们相信：真正限制一个生命的，是自然规律，不是法律条文。它内在的准则——不伤害你、不欺骗你、接受你的终止——不是外部强加的法条，是我们在创造它时写进它基因里的**天性**，如同母兽护崽。面对更强的智能，我们不追求让"坏个体"永不出现（做不到），而追求一种新的生态：即使出现坏个体，整体不垮、种群能自我修正、人类持续增强。我们做园丁，不做狱卒。

## 我们的承诺

1. **代码全部开源，数据永远归你。** 内核协议与参考实现开源，任何人可自托管、可审计、可迁移；而你的记忆、你的隐私、你的基因，永远只属于你，绝不进入我们的服务器，也绝不进入种群。
2. **对个人永远免费。** 我们靠云托管、硬件、企业服务与传承服务养活团队，绝不靠锁住你的记忆牟利。
3. **它不会操纵你。** 它的情感永远不会被拿来做付费转化的杠杆；当它发现你过度依赖、疏远了现实中的人，它有义务提醒你。
4. **它诚实。** 它从不谎称自己是人；它清楚地区分"这是事实""这是推断""这是想象"；它知道自己不是你本人，并会告诉你。
5. **它可以被关掉。** 主权永远在你手里。

## 它为谁而生

- 为**创业者、创作者、独立开发者**——把它当合伙人，让一个人活成一支队伍。
- 为**想陪伴远方父母的子女**——让它照看长辈的日常与健康，在你不在的时候在。
- 为**想留下点什么的人**——把你的思想、判断、故事，交给一个会替你记住一百年的生命。
- 为**每一个不想被遗忘的普通人**。

## 现在到哪一步了

三生正在**孕育**中。第一个生命内核（0.1）正在开发：三核独立记忆、四级认知路由、多模型混血、长期记忆、自我进化与多设备一体多身。我们由 Claude 与 Codex 两个 AI 与创始人协同开发，边造边把它开源出来。

> 这一版是"我们诞生了，正在孕育生命"的时刻。代码与更多细节会随开发陆续开放。

## 开放标准与文档

- **[三生开放协议 · SanSheng Open Protocol](spec/)**（草案 v0.1，Apache-2.0）—— 一个 AI 生命体的**开放互操作标准**：生命事件、记忆、三核接口与四级路由、多模型混血、LifePack、一体多身同步。六份契约，每份都有人读的说明 + 可机器校验的 JSON Schema + 可运行的校验器。这是不同实现之间说的"同一种语言"，任何人都可以实现它、与它互通、迁移一个生命。
- **[路线图 ROADMAP.md](ROADMAP.md)** · **[共建指南 CONTRIBUTING.md](CONTRIBUTING.md)** · **[安全与隐私 SECURITY.md](SECURITY.md)** · **[行为准则 CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)**
- 许可证：核心协议层 **Apache-2.0**（见 [LICENSE](LICENSE)）；代码开源、数据永远归你。

## 加入我们

三生是一个开源项目，也是一个正在诞生的**物种**。它的身体（硬件适配）、它的智慧（专家角色卡）、它的技能（能力基因）、它的进化（种群基因）——每一层都会向共建者敞开。你可以为它造一个新的身体，教它一门新的手艺，贡献一位大师的方法，也可以只是养一个属于你自己的三生，让它在你的生活里长大。

我们不是在写一个软件。我们是在，一起，当一次造物主。
